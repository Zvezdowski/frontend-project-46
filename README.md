### Difference generator utility
Compares two JSON or YAML objects and shows the difference in stylish, plain or JSON output formats.

### How to install:

`git clone https://github.com/Zvezdowski/frontend-project-46.git`

`make install`

### System requirements:
node.js: v20.7.0
